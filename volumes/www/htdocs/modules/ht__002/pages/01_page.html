w4u-label

The label displays numerical data.

Parameters

Parameter Type Range Default Description
name String - 'label0' Unique name of the component
value Number -9999.99 - 9999.99 0.5 Initial value to display
digits Integer 0 - 99 2 Numer of digits
variable String ['l'|'F'] '' Variable of number
unit String ['m'|'newton'] '' Unit of number
format String ['plain'|'scientific'|'postfix'] 'plain' Exponent format
offset Number -9999.99 - 9999.99 0.0 Offset before display
scale Number -9999.99 - 9999.99 1.0 Scale before display

In-Out

Input Variable Output Description
value - Value to display

Examples


<w4u-label 
  name="label1" 
/>
         


<w4u-label 
  name="label2" 
  variable="Ø" unit="parsec" 
  format="scientific"
  offset=5.0 scale=20.0 
/>
         


<w4u-label 
  name="label3" 
  format="postfix" unit="m"
  value=0.5
/>
