w4u-slider ()

The slider is a handsome possibility to modify a value in a range.

Parameters

Parameter Type Range Default Description
name String - 'slider0' Unique name of the component
orient String ['vertical'|'horizontal'] 'horizontal' Orientation of slider
color String ['#rrggbb'|'red'] 'green' Color of handle
background String ['#rrggbb'|'blue'] 'grey' Color of Range
min Number -9999.99 - 9999.99 0.0 Minimum value of output
max Number -9999.99 - 9999.99 1.0 Maximum value of output
continous Boolean [true|false] false Mode of value update during slide or only once when finished

In-Out

Input Variable Output Description
v Relative value of slider min-max

As the last example with slider3 shows, a slider can als be used to visualize input values

Examples

      

<w4u-slider 
  name="slider1" 
  orient="vertical" 
  color="#FF0000" 
  background="blue" 
  continous=true 
/>
         


<w4u-slider 
  name="slider2" orient="horizontal" 
  min=0.2 max=0.8 continous=false 
/>
         

<w4u-slider name="slider3" 
  orient="horizontal" background="#00FF00"
  min=-1.0 max=+1.0
/>
