w4u-map ()

The map shows geographical data.

Parameters

Parameter Type Range Default Description
name String - 'map0' Unique name of the component
latitude Number -90.0 - 90.0 0.0 Latitude of center of map
longitude Number -180.0 - 180.0 0.0 Longitude of center of map
zoom Number 0 - 16 '' Zoom level of map
width String ['80%'|'600px'] '90%' Width of the map
height String ['200px'|'400px'] '400px' Height of the map

In-Out

Input Variable Output Description
latitude Relative latitude
longitude Relative longitude
zoom Relative zoom level

Examples


<w4u-map 
  name="map1" 
  latitude="47.99" longitude="7.85"
  width="100%" height="250px"
/>
         


<w4u-label
