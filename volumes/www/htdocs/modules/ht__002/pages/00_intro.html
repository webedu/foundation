The key to dynamic content in webEDU are web components.

Those can be combined to interact which each other and fill a page with life.

Combining webEDU's web components

Source: Michael Kahle

In this module you learn...

how to use the web components of webEDU to create elements interacting with each other and the user.

Find out...

  • ... how components can prompt for user input data.
  • ... how components can process data.
  • ... how components can display and visualize output data.
  • ... how these components can be connected by data streams.

Prepositions:

You should be familiar with the basics of creating modules. Work throught the module How to implement new modules first.

Be sure to have some knowledge about HTML5.

Estimated time needed: about 30 minutes

Autor:

Michael Kahle (WEBGEO|Docs), Institut für Physische Geographie, Universität Freiburg.
