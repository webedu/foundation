Interactive modules can help to easier understand complicated facts.

With the help of animations, elements with interaction, dynamic input, output and visualization of values complex relations get tangible. Step wise progress and active recapitulations improve the learning process in webEDU.

Implement your first own module

Source: No citation needed

In this module you learn...

how to write your own modules. Writing your own modules with interactive elements is nearly as simple as writing plain HTML code.

Find out...

  • ... how to set up the initial file structure to create a module with multiple pages.
  • ... how to format the single pages.
  • ... how to add your first interactive components.

Prepositions:

Have a look to some existing modules, i.e. Adiabatische Prozesse. Or just take this very module as example to see whats possible.

In addition, it can help, to have some basic knowledge about HTML5 and Bootstrap. Or learn it step by step while implementing your first module.

Estimated time needed: about 30 minutes
